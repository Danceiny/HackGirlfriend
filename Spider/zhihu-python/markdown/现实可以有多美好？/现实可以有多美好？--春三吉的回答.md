# 现实可以有多美好？
## 作者: 春三吉  赞同: 248
那天凌晨家里被偷了。。。  
  
等晚上挤上公交才发现小偷居然连公交卡都偷了！！！  
  
车上人很多，我站在门口翻遍个整个包的所有夹层，可惜钱包里只有刚刚取的整钱。急得我满头大汗！  
  
然后站在我旁边的美女帮我刷卡啦！  
  
然后我说美女我支付宝把钱还给你吧，她说不用，我坚持！  
  
然后我就 get 到美女的手机号啦！
