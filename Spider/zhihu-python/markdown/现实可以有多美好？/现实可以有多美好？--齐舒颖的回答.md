# 现实可以有多美好？
## 作者: 齐舒颖  赞同: 6
其实不是什么特别的事。  
从小爸妈在隔壁的村子开鱼丸店，我和哥哥在由奶奶带着，爸妈大概两三天回来一次，但从来不在家里住。
小时候放假或周末我就跑去爸妈的店里，和他们一起睡，挤在他们中间睡觉超级开心。  
但是有一次，爸妈回了家，然后晚上要回去店里，我那个时候年纪小，很想很想和爸妈走，不想他们离开，但是第二天要上学，没办法去，那个晚上我哭的几乎撕心裂肺，爸妈还
是走了。  
后来上高中的一年春节，爸妈那时已经换了工作，春节不用再做鱼丸，春节前后的几个晚上，我和哥哥爸妈一起偎在床铺上看林正英的僵尸片，结果爸爸和哥哥都也被吓到了。  
我有记忆以来我们一家从来没有在一起轻松无虑的看过电视或做其他事情。  
所以那一刻，从前所有的不理解和难过都烟消云散，只觉得世界美好。  
在一起，从来都是最美好的。
